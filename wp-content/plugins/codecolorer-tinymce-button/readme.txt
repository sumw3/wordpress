== Description ==

This plugin was developed for another plugin - CodeColorer (http://wordpress.org/extend/plugins/codecolorer/), but easily can be rewrited to another syntax highlighting plugin.

I don't know why, but not one plugin for syntax highlight haven't own button on TinyMCE panel.

NOTE: It is important that folder of plugin was named as "codecolorer-tinymce-button" 

== Installation ==

1. Install CodeColorer (http://wordpress.org/extend/plugins/codecolorer/)
2. Upload plugin folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use new button in TinyMCE panel
